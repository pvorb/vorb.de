Markdown

Dieses Dokument bietet einen Überblick über die gängigsten Auszeichnungselemente in Markdown. Weitere Informationen gibt es auf der Website des Autors.

Formatierung


    **fett**, _kursiv_, -durchgestrichen-

Links


[Wikipedia](http://de.wikipedia.org)

<https://vorb.de>

wird zu

Wikipedia

https://vorb.de

Bilder


![Alternativtext](https://vorb.de/res/flower.jpg)

wird zu

Zitate

Text, der mit > eingerückt wird, wird als Zitat dargestellt.


> Von den meisten Büchern bleiben nur Zitate übrig.
> Warum nicht gleich nur Zitate schreiben?
‒ Stanisław Jerzy Lec

wird zu

Von den meisten Büchern bleiben nur Zitate übrig. Warum nicht gleich nur Zitate schreiben?

‒ Stanisław Jerzy Lec

Listen

Listen können sowohl mit Hilfe von *, + und - als auch durch Zahlen erstellt werden.


*   Hund
*   Katze
*   Maus

1.  Montag
2.  Dienstag
3.  Mittwoch

wird zu

  • Hund
  • Katze
  • Maus
  1. Montag
  2. Dienstag
  3. Mittwoch

Quelltext

Alles, was durch vier Leerzeichen eingerückt ist, wird als Quelltext dargestellt.


    // add a to b
    function add(a, b) {
      return a + b;
    }

wird zu


// add a to b
function add(a, b) {
  return a + b;
}
