title: Bread


Bread is my own CMS/blog software/static website generator.

*   It’s written in JavaScript.
*   It’s built on top of [Node.js](http://nodejs.org/)
*   It’s built on top of [Pandoc](http://johnmacfarlane.net/pandoc/)
*   It’s build on top of [MongoDB](http://www.mongodb.org/)

You can see the source code at [the Github
repository](https://github.com/pvorb/node-bread).
